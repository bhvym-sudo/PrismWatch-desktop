# PrismWatch-desktop
- PrismWatch is a forensic tool to dive into the behaviour of android apps
- We are designing it to:
 
    - monitor the logs of app.
    - extract the metadata such as permissions and activity
    - network activity.
    - background activity
    - integrated shell which can be used to view the third party logs

# ⚠️REQUIRES ROOTED PHONE WITH USB DEBUGGING
